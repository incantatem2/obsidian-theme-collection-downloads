# How to install:
1. Open your explorer and access the obsidian vault in which you want the theme.
2. Open the **.obsidian** folder which should be on top.
3. Open the "**themes**" folder.
4. Place the .zip in that folder.
5. Unzip (restart obsidian.md if the theme does not show up instantly).
6. Done, the theme should now appear in the list of your installed themes.
